# AI 주식 자동매매 시스템

이 프로젝트는 한국투자증권 모의투자 API를 사용하여 AI 기반 주식 자동매매를 수행하는 프로그램 뼈대입니다.

## 폴더 구조 (어떤 목적으로 나뉘어 있는지)
- `config/`: 프로그램 설정과 비밀번호(API 키) 등 민감한 정보를 관리하는 폴더
- `src/`: 실제 코드가 들어있는 메인 폴더
  - `api/`: 한국투자증권과 직접 통신하여 가격을 가져오고 주문을 넣는 코드
  - `strategies/`: 언제 사고 팔지 결정하는 알고리즘/AI 전략 코드
  - `utils/`: 실행 기록(로그)을 남기는 등 시스템 운영을 돕는 코드
  - `main.py`: 이 프로그램이 시작되는 출발점
- `requirements.txt`: 프로그램 실행에 필요한 추가 도구 목록

## 시작하는 방법
1. `config/.env.example` 파일의 이름을 `.env`로 바꾸고 발급받은 모의투자 API 키를 입력하세요.
2. 터미널(명령 프롬프트)에서 아래 명령어를 쳐서 필수 도구들을 설치하세요.
   ```bash
   pip install -r requirements.txt
   ```
3. `src/main.py`를 실행하여 프로그램을 켭니다.
   ```bash
   python src/main.py
   ```
